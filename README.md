## Tech Stack

| Layer        | Technology                                      |
|------------|--------------------------------------------------|
| Frontend   | Vite + React 18 + TypeScript
| Backend    | Node.js + Express + TypeScript                   |
| Database   | MongoDB (with Mongoose ODM)                      |                              |
| Deployment | Docker + Docker Compose (multi-container)        |
| Dev Tools  | ESLint + Prettier + lint-staged          |

## Features

- Lightning-fast development with **Vite**
- Clean **React + TypeScript** frontend with `@/` alias
- Secure **Node.js + Express** API with Zod validation
- Full **JWT authentication** (login, register, protected routes)
- MongoDB integration with proper indexing
- Production-ready **Docker + Docker Compose** setup
- Hot reload for both frontend and backend
- Environment-based configuration (`.env`)
- Error handling & logging
